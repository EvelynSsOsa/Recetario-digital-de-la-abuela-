# Recetario-digital-de-la-abuela-
Aplicación web desarrollada en Python que permite crear y organizar recetas de manera sencilla y muy intuitiva.<br><br>  

🍒Descripción: <br> 
Esta applicación funciona como un recetario digital donde los usuarios pueden:<br>

🧄Crear diferentes categorias(por ejemplo: postres,ensaladas,pastas... o lo que tú desees).<br>
🧄Agregar recetas dentro de cada categoria.<br>
🧄Organizar sus recetas de forma clara y accesible.<br><br>
🧄Cada receta se almacena dentro de su respectiva categoría,facilitando la navegación y gestión de la información.<br><br> 

Tecnologías utilizadas:<br>  
🥬Python.<br> 
🥬Streamlit(para la interfaz gráfica del usuario).<br> 
🥬Pathlib.<br><br>
