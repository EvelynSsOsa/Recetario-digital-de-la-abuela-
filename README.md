# 📖 Recetario Digital de la Abuela

Aplicación web desarrollada en **Python** que permite crear y organizar recetas de manera sencilla e intuitiva.

---

## 🍒 Descripción

Esta aplicación funciona como un **recetario digital** donde los usuarios pueden:

- 🧄 Crear diferentes **categorías** (postres, ensaladas, pastas, bebidas, etc.)
- 🧄 **Agregar recetas** dentro de cada categoría
- 🧄 **Organizar** sus recetas de forma clara y accesible

Cada receta se almacena dentro de su respectiva categoría, facilitando la navegación y gestión de la información.

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Descripción |
|------------|-------------|
| 🐍 **Python** | Lenguaje de programación principal |
| 🎈 **Streamlit** | Framework para la interfaz gráfica |
| 📁 **Pathlib** | Manejo de rutas y archivos |
